## Project Overview

创建一个个人健康管理系统，用于追踪和分析用户的健康数据。

### 核心功能模块

1. **运动追踪**
   - 记录运动类型、时长、消耗卡路里
   - 运动频率统计
   - 运动目标设定和达成分析

2. **饮食管理**
   - 记录每日饮食
   - 热量摄入统计
   - 营养均衡分析

3. **睡眠监测**
   - 睡眠时长记录
   - 睡眠质量评估
   - 睡眠模式分析

4. **体重/体脂追踪**
   - 体重变化曲线
   - 体脂率记录
   - 身体指标趋势分析

5. **报告生成**
   - 周报/月报/年报
   - 健康趋势分析
   - 改进建议生成
