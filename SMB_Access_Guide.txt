## Windows Access

### Method 1: Direct SMB Access
1. Press `Win + R`
2. Type: `\\172.17.0.2\health-management`
3. Press Enter

### Method 2: File Explorer
1. Open File Explorer
2. Address bar: `\\172.17.0.2\health-management`
3. Press Enter

### Method 3: Map Network Drive
1. Right-click "This PC"
2. Select "Map network drive"
3. Drive: `Z:`
4. Folder: `\\172.17.0.2\health-management`
5. Done!
